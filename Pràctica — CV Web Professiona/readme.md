Este es mi CV, que he creado usando HTML y CSS como parte de la práctica del ciclo de ASIX en La Salle Gràcia.

El diseño es sencillo, con tonos grises que combinan bien. El contenido principal está en un recuadro blanco en el centro, mientras que los lados quedan grises, dando la sensación de una tarjeta centrada en la pantalla.

El proyecto incluye un archivo HTML con toda la información, un CSS con los estilos y una carpeta con mi fotografía. He usado Flexbox para que se vea bien tanto en móvil como en ordenador, y variables CSS para no repetir los mismos colores.

El CV tiene las secciones habituales: mi foto y datos de contacto, una presentación personal, tres proyectos destacados , mi formación actual, habilidades y los idiomas que hablo.
